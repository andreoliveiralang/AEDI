Lista dinâmica

Lista dinâmica para dados que não sabemos a quantidade exata de dados.
Vetor caso soubermos a quantidade.
Listas, como vimos no exemplo de lanches em python.

Para trabalhar om lista dinâmica tem que importar:
-	import java.util.ArrayList;
-	import java.util.List;

size() é a mesma coisa de lenght – nada mais é que o tamanho do vetor

para adicionar na lista, em posição final da lista,  usamos: 
LISTA.add("Ricardo");
Para adicionar na lista em posição especifica ,usamos:
LISTA.add(2,nome);
Vai ser inserido na posição 2
Dica professor - Colocar sempre no final.

Para atualizar um valor usar o comando .set
LISTA.set(1,"Pedro");
